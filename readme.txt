Для работы этой версии сайта необходимы следующие компоненты/программы:
1.Node js и сопутствующие библиотеки
2.PostgreSQL

Установка:
1.Скачиваем и устанавливаем Node js c официального сайта через устрановщик (https://nodejs.org/en)
2.Скачиваем и устанавливаем PostgreSQL c официального сайта через устрановщик (https://www.postgresql.org/)
3.Открываем директорию с сайтом через редактор кода и устанавливаем необходимые библиотеки:
3.1. Командой "Node -v" можно проверить правильность установки Node js
3.2. Командой "npm i" устанавливаем недостающие библиотеки
4. В меню пуск находим программу pgAdmin4 и запускаем её, после чего в открывшемся окне вводим пароль,
который установили при установке PostgreSQL. 
4.1. Далее необходимо создать новую базу данных через правый щекчок по серверу PostgreSQL -> Create -> Database 
и в открывшеммся окне в поле Database необходимо ввести название базы данных (photocenter)
5. Нужно скорректировать данные о базе данных в файле Index.js. Для этого необходимо отктыть файл Index.js и найти в нем строчку:
const sequelize = new Sequelize('postgres://postgres:Pass@localhost:5432/photocenter');
В этой строчке необхомо заменить "Pass" на пароль, введённых при установке PostgreSQL и, если при создании базы данных 
было введено название отличное от "photocenter", то необходимо заменить его на название базы данных.
6. Необходимо произвести создание таблиц в базе данных. Для этого необходимо произвести предварительный запуск сайта rомандой - node index.js
7. Запускаете сайт командой "node Index.js" и открываете в браузере адрес "http://localhost:3000/" 

Всё готово, сайт работает.

Проверить его работу можно в программе pgAdmin4 открыв вкладку с базой данных, следом вкладку Shemas, следом вкладку Public, следом владку Tables.
Далее необходимо правой кнопкой мыши нажать на таблицу Records и выбрать пункт View/Edit data -> All Rows, после чего откроется окно, в котором можно будет увидеть созданные записи.
